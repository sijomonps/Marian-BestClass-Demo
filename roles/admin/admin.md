# Admin Role

## Responsibilities
- Manage simple system setup for all roles
- Manage criteria and marks
- Manage users and departments
- Control submission and evaluation settings

## Features
- Dashboard
	- Total students
	- Total submissions
	- Pending verifications
	- Submission status (ON/OFF)
- Criteria Management
	- Add category
	- Add item under category
	- Set marks and type (fixed / count / range)
	- View grouped list
- User Management
	- Add user with Name, Role, Department
	- View simple user table
- Department Management
	- Add department
	- View and delete department list (delete blocked when in use)
- Settings
	- Submission ON/OFF
	- Evaluation ON/OFF
	- Academic year selector and activate action
	- Reset demo data
