# System Workflow

## Step 1: Submission
- Student selects category
- Adds activity
- Uploads proof
- Submits

## Step 2: Teacher Verification
- Approve / Reject / Request correction

## Step 3: Evaluation
- Evaluator verifies
- Assigns marks
- Uses auto calculation or manual override

## Step 4: Scoring
- Marks are summed
- Normalized based on number of students

## Step 5: Ranking
- Classes are ranked
- Grades assigned
