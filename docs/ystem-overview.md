# Best Class Evaluation System - Overview

## Purpose
This system is designed to evaluate and rank classes based on student activities in a structured and fair way.

## Core Idea
- Students submit activities with proof
- Teachers verify submissions
- Evaluators assign marks based on predefined criteria
- System calculates final score
- Classes are ranked fairly

## Key Features
- Role-based system
- Multi-level verification
- Criteria-based scoring
- Automatic mark calculation
- Dashboard analytics
- Ranking system

## Roles
- Student
- Teacher
- Evaluator
- Admin
- HOD / IQAC
