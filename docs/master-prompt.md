# MASTER PROMPT - Best Class Evaluation System

You are building a full frontend system for a college evaluation platform.

## System Summary
A role-based system where:
- Students submit activities
- Teachers verify
- Evaluators assign marks
- System calculates scores and ranks classes

## Roles
Student, Teacher, Evaluator, Admin, HOD/IQAC

## Key Features
- Checklist-based student UI
- Single action evaluation (Verify & Save)
- Criteria-based scoring
- Category-based system
- Ranking and analytics

## UI Principles
- Simple for beginners
- Minimal design
- Clear actions
- No complex graphs

## Important UX Decisions
- Checklist instead of charts
- Single button actions
- Role-based pages
- Clean navigation

## Tech Stack
- HTML, CSS, JS
- No backend (prototype)

## Goal
Build a clean, simple, and realistic evaluation system.
